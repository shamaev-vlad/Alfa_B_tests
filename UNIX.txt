Задания:
1) Узнать имя пользователя, под которым осуществлен вход в систему
Ответ: whoami или echo $USER
2) Перейти в домашнюю директорию
Ответ: cd
3) Посмотреть содержимое директории
Ответ: ls
4) Создать файл с именем test
Ответ: touch test
5) Заполнить файл строкой echo Hello Oleg!
Ответ: cat >> test Hello Oleg! или vim test
6) Посмотреть права доступа на файл test
Ответ: ls -l test
7) Добавить права на выполнение себе, на чтение+запись для группы, остальным запретить все
Ответ: chmod u+rwx;
       chmod g+rw;
       chmod o-rwx
8) Выполнить файл test
Ответ: sh test
9) Переименовать файл в test2
Ответ: mv test test2
10) Удалить файл test2
Ответ: rm test2
